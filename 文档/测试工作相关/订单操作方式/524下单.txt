CRM首页 -> 点击"客户"按钮 -> 查询 -> 延期鉴权 -> 关闭弹出窗口 -> 查询 -> 输入一个客户标识:202111172812 -> 执行 -> 点击"客户业务受理"链接 -> 在右侧"业务流程"的"销售品新装"点击"设置" -> 在"客户地址"栏中如果没有地址信息，则点击"获取地址"按钮，获取最新的地址 -> 点击"下一步" -> 在"销售品列表"栏中点击"查询" -> 在"销售品编码"框中输入"*524"或者"100524"，以找寻后付费宽带产品 -> 点击"下一步" ->检查受理的产品名称和客户标识等信息，是否正确，然后点击"完成"，开始受理订单 -> 在订单信息界面，往下拉在行项目上，找到"产品配置" -> 接入方式可选择"FTTH"或者"ADSL" -> 在"其他业务"标签栏下，找到"爱BABY（专业版）"子产品,在下拉列表的项目列表上选择"子产品",在旁边的数量上填指定数目(同理"爱BABY（标准版）"子产品)，并点击"添加项目"按钮，以完成子产品数量的添加 -> 将页面拉到最上方，点击"完成"按钮，以完成产品配置 -> 再次将拉到页面上方，点击"合法性检验"，对订单进行检验 -> 当订单状态从"新建"变为"通过合法性校验"，就可以提交订单了-> 提交订单完成后 -> 查询订单号 -> 查看订单状态为"IBP已收到" 则可前往P7上查询订单状态

在P7上查询订单状态: 在Query -> 输入CRM订单号 -> 查看订单状态 -> 当流程出现"zz服务配置子流程",则可以找综资的人员对订单进行后续的配置
